About Solution

What Is It?

FourSight: A deployable, flexible, and reliable Predictive Maintenance.

  • FourSight is a Machine Learning service used to integrate Predictive Maintenance into any environment
  • The use of Predictive Maintenance in Industrial Manufacturing brings immense cost and time saving
  • FourSight allows users to detect failures and required maintenance before they occur
Request a Demo!
Our Unique Solution

Why FourSight?

  • FourSight differs from other Predictive Maintenance solutions by taking the user complexity out of the question
  • Traditional Predictive Maintenance solutions require trained Machine Learning engineers to develop models to fit each machine data point
  • FourSight uses Large Language Model algorithms to detect data fields and values which can indicate potential issues automatically. Then using Azure AutoML, FourSight allows the data to dictate the most suitable model for detecting failures early
How it Works

User Interaction

  • When using FourSight, users are greeted with a dashboard indicating overall factory health status as well as any major problems that should be addressed.
  • FourSight sets priority levels based on each predicted failure allowing the most important issues to be fixed first
  • FourSight breaks machines down, where each Machine can have multiple Components and each component can have multiple Sensors. This allows users to track maintenance requirements to specific machine components reducing maintenance time.
Behind the Scenes

Data Ingress

  • Data ingress is handled by Azure Event Hubs. Event Hubs can handle different protocols such as AMQP, HTTPS or Apache Kafka. Sending sensor data via these protocols can be handled in multiple ways
  • If the sensors are connected to a computer for data logging, a simple service can be run on the computer to forward the data points to FourSight
  • In other scenarios such as sensors without a computer, an IoT Gateway can be used to forward the data. This allows sensors to transmit data using LAN, WiFi or Cellular
